## Install

```sh
npm install
```

## Usage

```sh
npm start
You have to set up your own .env
```
## Technologies

```sh
MongoDB
Express
React
Node.js
Redux
JWT / Passport
Joi
Sessions
Paypal and MercadoPago integration
```
